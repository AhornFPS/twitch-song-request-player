Credential Guide

What you actually need

Required for Twitch chat requests

Fill in Twitch channel, Bot username, and Bot OAuth token. This is enough for `!sr`, `!skip`, `!delete`, `!save`, and `!currentsong`.

Built-in Twitch login path

The desktop app uses the bundled public Twitch Client ID for in-app bot login. End users do not need to type a Twitch Client ID or Client Secret into the GUI.

Optional YouTube search key

YouTube API key is only needed for `!sr artist song name`. If viewers request direct links, you can leave it empty.

Category Rules

Suppress by Twitch category

Add categories that should mute chat announcements or stop music playback entirely while your stream is in that category.

Suppress chat messages

The bot will stop sending chat replies and now-playing messages while the current Twitch category matches one of these entries.

Suppress music entirely

The player will stop the active track and hold queued playback until the stream leaves one of these categories.
